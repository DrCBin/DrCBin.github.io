```python

class Student:
    def __init__(self, name, age):
        self.name = name
        self.age = age

>>> stu = Student('Tom', 40)
>>> stu.name
Tom
>>> stu.name = 'Jim'
>> stu.name
Jim
```

通过上面的例子，可见，是可以任意更改实例对象的属性的,但是这样就失去的封装的意义,所以我们要对类外访问类成员加以限制

```python

class Student:
    def __init__(self, name, age):
        self.__name = name
        self.__age = age

>>> stu = Student('Tom', 40)
>>> stu.__name
^*&)报错
```
 这样，通过在属性名前面加上`__`就能限制该属性为私有的了.

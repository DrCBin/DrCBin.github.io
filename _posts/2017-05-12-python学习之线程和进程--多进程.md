# 日后补上多线程和多进程编程,还有什么线程池，等待唤醒等.这些廖雪峰的博客都没有, 还有异步IO和分布式进程


- 多线程对基本类型变量的操作无效,原因是pytho的变量作用域: `class`, `def`, `lambda`

```python
a = 5  # 一个基本类型变量

def change():
    a = a + 3
    print(a)

>>> change()  # 报错


def change():
    global a  # 申明一个全局变量
    a = a + 3
    print(a)

>>> change()  #不会错
8   # 结果为8
```

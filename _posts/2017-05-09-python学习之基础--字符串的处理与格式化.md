### 分割字符串 ###

1.使用切片

```python3
>>> s = 'abcde'
>>> s[:2]
'ab'    # 左起为0,右边是开区间
>>> s[-3:]
'cde'    # 从右边起第一个为-1,-2,-3...
```

2.使用分割函数:`str.split`

```python3
'''
str.split(sep=None, maxsplit=-1)

function: 正向分割字符串

arguments:
    sep: 分割的标志
    maxsplit: 分割的次数

return:
    list of str
'''
>>> s = '1.2.3.4.5.6.7'
>>> s.split('.')
['1','2','3','4','5','6','7']

>>> s.split('.', 1)
['1', '2.3.4.5.6.7']
```


3.使用分割函数:`str.rsplit`

```python3
'''
str.rsplit(sep=None, maxsplit=-1)

function: 反向分割字符串

arguments:
    sep: 分割的标志
    maxsplit: 分割的次数

return:
    list of str
'''
>>> s = '1.2.3.4.5.6.7'
>>> s.rsplit('.')
['1','2','3','4','5','6','7']

>>> s.rsplit('.', 1)
['1.2.3.4.5.6', '7']
```

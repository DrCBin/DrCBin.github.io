迭代，依次遍历访问可迭代对象中的元素。

可迭代对象的判断:

```python
from collections import Iterable
L = []
instance(L, Iterable)
```

如何像其他语言一样访问可迭代对象的下标:使用`enumerate`函数

```python
for i, value in enumerate(['A', 'B', 'C']):
    print(i, value)
```

这样就达到了访问值的同时访问下标的效果.

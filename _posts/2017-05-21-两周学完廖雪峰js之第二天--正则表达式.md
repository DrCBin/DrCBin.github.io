# 第二天

-------


js中创建正则表达式的两种方式:

```js
var re = /^\d{3}\-\d(3,8)$/;    // 用'/..../'包裹起来

var re2 = new RegExp('^\\d{3}\\-\\d(3,8)$'); // 通过创建对象的方式创建,注意'\'需要转义
```

利用re分割字符串

```js
'a b    c'.split(' '); // ['a', ' ', ' ', 'c'],无法识别多个空格
'a b    c'.split(/\s+/);
```

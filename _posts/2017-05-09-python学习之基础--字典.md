# 字典(dict) #

字典，类似于java的map， 其他地方的键值对, 就是键和值一一对应，给需要的键，就可以得到需要的值

```python
# 一个关于学生成绩的字典
grades = {'marry': 91, 'tom':80, 'jim':100}

# 获取tom的成绩
>>> grades['tom']
80
```

其中的名字叫键， 成绩叫值。

当访问一个不存在的键时会报错

```python
>>> grades['小明']
(*&(*HJ(*&  报错了
```

所以在访问之前需要判断

```python
if '小明' in grades:
	grades['小明']
```

还可以通过调用函数的方式访问

```python
>>> grades.get('Tom')
80

>>> grades.get('小明')
(&*^(* 报错

# 可以设置在获取失败的时候给默认值
>>> grades.get('小明', -1)
-1
```


也可以删除某个键值对

```python
grades.pop('tom')
```
